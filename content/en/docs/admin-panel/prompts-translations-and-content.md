## Prompts Management

1. Open Prompts.
2. Select category.
3. Select subcategory.
4. Open target prompt template.
5. Edit text in editor.
6. Validate syntax.
7. Save and verify version update.

## Prompt Quality Checks

1. Confirm variable placeholders are valid.
2. Review lint/syntax errors.
3. Review diff between previous and current versions.

## Translations Management

1. Open Translations.
2. Filter by namespace and locale.
3. Search by key or value.
4. Edit existing translation.
5. Create missing translation keys where needed.
6. Save and validate in target UI route.

## Mentor Personality and Interests

1. Open Mentor Personality.
2. Edit localized mentor fields and expertise arrays.
3. Save and verify list updates.
4. Open Interests module.
5. Review status queue and update where allowed.

## Common Questions

Q: Prompt save fails with template error.
A: Fix invalid placeholder/template syntax and retry.

Q: Translation saved but UI unchanged.
A: Confirm correct namespace/locale and clear stale cache.
