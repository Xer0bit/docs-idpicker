This guide covers the parent child-detail page, journey-map outputs, and advisor chat context.

## A) Open Student Detail

1. From parent dashboard, click linked student card.
2. Open `/dashboard/{studentId}` page.
3. Confirm student header (name/avatar/basic profile) appears.

## B) Review Progress and Insights

In student detail, review:

- Assessment progress and completion indicators
- Recommendations summary
- Mentor and advisor activity
- Premium/subscription status context

## C) Journey Map Preview (Click-by-Click)

1. Open Journey Map section.
2. Select data point filters:
   - demographics
   - education
   - psychological
   - career
   - social
   - family
   - health
3. Click Preview.
4. Review returned structured preview.

## D) Journey Map Download

1. Keep selected data points.
2. Click Download/Generate Journey Map.
3. Wait for map link generation.
4. Open generated map link in browser.

## E) Advisor Chat Indicators and Access

1. Locate advisor chat panel/button in student detail.
2. Check unread count badge.
3. Open parent-advisor chat if available.
4. Read thread and send message if enabled in panel.

## F) Subscription Plans View

1. Open subscription area on child detail.
2. Review available plans and selected plan state.
3. Continue purchase flow when enabled in deployment.

## Common Parent Questions

Q: Journey map preview is empty.
A: Ensure at least one data point is selected.

Q: Download generated but no content.
A: Retry with fewer data points first, then regenerate.

Q: Advisor unread count is high but no new message appears.
A: Refresh chat endpoint and confirm active student context.
