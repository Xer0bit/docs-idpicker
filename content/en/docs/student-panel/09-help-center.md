Help Center is where students create support tickets and track responses.

## Open Help Center

1. In sidebar, click Help Center.
2. Wait for page to load tabs.
3. Confirm New Ticket and My Tickets tabs are visible.

## Create Ticket (Click-by-Click)

1. Ensure New Ticket tab is selected.
2. Click Subject input and type issue title.
3. Click Category dropdown and select category.
4. Click Priority dropdown and select urgency.
5. Click Description and write detailed issue context.
6. Click Submit.
7. Wait for success confirmation.

## Category and Priority Guidance

Category:

- Technical, Account, Billing, Feature, General, Other

Priority:

- Low, Medium, High, Urgent

Tip:

- Use High/Urgent only for blockers.

## Track Existing Tickets

1. Click My Tickets tab.
2. Open list of your submitted tickets.
3. Review each ticket status and timestamps.

Status meanings:

- Pending: created, waiting for handling
- In progress: currently handled by support
- Resolved: solution provided
- Closed: finalized/archived

## Write Better Tickets

Include:

- Exact page/module name
- Exact error message text
- Reproduction steps (what you clicked)
- Expected vs actual behavior

## Common Student Questions

Q: Submit button shows validation error.
A: Subject and Description are required.

Q: I submitted but ticket list did not refresh.
A: Switch to My Tickets tab or reload page.

Q: Which category for login issue?
A: Start with Account unless billing/payment is involved.
