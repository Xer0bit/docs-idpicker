1 - Overview

What IDPicker does and how users move through the platform

IDPicker helps students evaluate strengths, explore academic and career options, and get guided recommendations.

Primary User Roles

  • Student
  • Advisor
  • Parent

Panel Entry Points

  • Student Panel: main user learning and recommendation workspace
  • Advisor Panel: supervision, student support, and sub-advisor administration
  • Parent Panel: visibility for linked students

Student Account Types

  • Linked Student: account linked to parent visibility model
  • Individual Student: independent account with direct access only

Purpose URL

URL Validation Checklist

  1. Login URL is reachable.
