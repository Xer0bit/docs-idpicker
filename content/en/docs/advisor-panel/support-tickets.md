Support Tickets page helps advisors track and resolve student-reported issues.

## A) Open Support Tickets

1. In sidebar, click Support Tickets.
2. Wait for ticket list and summary cards.

## B) Filter and Prioritize

1. Filter by status (pending, in_progress, resolved, closed).
2. Filter by priority (low, medium, high, urgent).
3. Sort list for triage order.
4. Open most urgent unresolved tickets first.

## C) Review Ticket Details

1. Click ticket row/card.
2. Read subject and full description.
3. Check user details and timestamps.
4. Identify category and visibility context.

## D) Update Ticket Status

1. Open status control.
2. Set new status (for example in_progress).
3. Save update.
4. Confirm status badge changed in list.

## E) Add Resolution Message

1. Open resolution input.
2. Write clear resolution notes.
3. Save and mark resolved when done.
4. Close ticket when finalized.

## Common Advisor Questions

Q: Status changed but not persisted.
A: Reopen ticket and verify update request succeeded.

Q: Too many tickets in one view.
A: Use status and priority filters together for triage.

Q: Student says issue still unresolved.
A: Reopen ticket and move status back to in_progress.
