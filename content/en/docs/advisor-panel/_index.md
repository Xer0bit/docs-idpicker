Use this section to operate advisor workflows from login to student supervision.

## Core Responsibilities

- Monitor student outcomes and assessments
- Review recommendations and reports
- Manage support workflow
- Create and manage sub-advisors

## Guide Sequence

1. Advisor login and dashboard
2. Student assignment and progress review
3. Sub-advisor creation and permission control
4. Parent-linked visibility validation

## Sub-Advisor Capability

Advisors can create sub-advisor accounts to delegate supervision across student cohorts.

### Recommended Sub-Advisor Fields

- Full name
- Email
- Assigned student group
- Permission scope
- Active status

### Validation Checklist

- Sub-advisor can log in with assigned credentials
- Sub-advisor sees only authorized students
- Parent-linked visibility respects permission model
- Deactivating sub-advisor revokes access immediately
