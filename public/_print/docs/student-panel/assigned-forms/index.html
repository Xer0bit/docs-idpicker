Assigned Forms

Detailed click-by-click assigned forms walkthrough

Assigned forms are organization/advisor forms that students complete from dashboard or direct link.

Open Assigned Form from Dashboard

  1. Open Dashboard.
  2. Locate Assigned Forms block.
  3. Click Open Form on pending item.
  4. Wait for form page to load.

Understand Form Header

At top, you may see:

  • Organization banner image
  • Form title
  • Form description
  • Editability state

Complete Form Fields (Click-by-Click)

For each field:

  1. Read label and required indicator.
  2. Click input control.
  3. Enter/select response.
  4. Continue to next field.

Supported field patterns include text, textarea, number, email, dropdown, radio, checkbox, date/time/datetime, and rating.

Submit Response

  1. Scroll to submit action.
  2. Click Submit.
  3. Wait for success message.
  4. Return to dashboard.

If Form Is Read-Only

If already submitted and editing is disabled:

  • Inputs are locked.
  • You can review previous response but cannot modify.

Common Student Questions

Q: I submitted but cannot edit now. A: Form is configured with no post-submit editing.

Q: Required field warning appears. A: Fill all required fields marked with required indicator.

Q: I cannot see one field my friend has. A: Field set depends on assigned form configuration.
