## Student Management

1. Open Student Management.
2. Use search by name/email.
3. Apply filters and sorting.
4. Expand student row for full detail.
5. Review verification, premium, and activity counters.
6. Use delete action only after confirmation review.

## Parent Management

1. Open Parent Management.
2. Search/filter parents.
3. Open parent detail.
4. Edit parent profile fields if needed.
5. Link parent to student or unlink relationship.
6. Send invite when required.

## Organization Management

1. Open Organization Management.
2. Click Create Organization.
3. Fill organization profile and primary admin details.
4. Save and verify organization appears in list.
5. Edit, export, or delete based on policy.

## Safety Checklist Before Deletion

1. Confirm no active dependencies.
2. Export relevant records if needed.
3. Use confirmation dialogs carefully.

## Common Questions

Q: Parent linked but student still missing.
A: Refresh relation list and verify student account exists in same environment.

Q: Organization created without expected admin access.
A: Reopen organization record and verify admin assignment fields.
