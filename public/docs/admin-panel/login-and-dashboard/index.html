Admin Login and Dashboard

How to access the super-admin portal, review global KPIs, and monitor platform health

The Admin Panel is the highest level of access in IDPicker, designed for platform owners and high-level operations staff. This guide covers secure access and daily monitoring routines.

1. Secure Admin Login

Accessing the admin portal requires navigating to a specific, protected route.

  1. Open the Admin Login URL (often /admin/login).
  2. Enter your authorized admin Email or Username.
  3. Enter your Password.
  4. Click Sign In.
  5. Once your high-level authorization cookie is validated, you are routed directly to the global dashboard.

Forgotten Password? If you lose your password, click Forgot Password, enter your email, and use the secure reset link sent to your inbox. Because of the high privilege level of this account, password strength requirements are strictly enforced.

2. Daily Dashboard Checks

The admin dashboard provides a live pulse of the entire platform’s health and activity. Make it a habit to review these four areas every time you log in:

  1. Global KPI Cards: Check total active users, new registrations today, and total completed assessments.
  2. Recent Assessments Table: A live feed of the newest data coming into the system from across all organizations.
  3. Registration Trend Chart: Look for sudden spikes or drops in daily sign-ups.
  4. System Activity Panel: A generalized event feed tracking high-level system actions.

3. Monitoring AI Token Usage

Because the AI Mentor and generation features consume API tokens, monitoring usage is critical for cost control.

  1. Scroll to the Token Usage section on the dashboard.
  2. Adjust the date range filter to look at the last 24 hours or the last 7 days.
  3. Review the charts for Input Tokens (prompts sent to the AI) and Output Tokens (responses generated by the AI).
  4. If you notice a massive, unexplained spike in usage, immediately cross-reference the exact timestamp with the Realtime Logs to identify potential abuse or infinite loops.

Common Questions

Situation What to do
I log in successfully, but the dashboard widgets are completely blank High-level data queries can take a moment to resolve. Wait 5 seconds and refresh the page
I click a link and keep getting bounced back to the login page Your secure session cookie has likely expired. Clear your browser cookies for the domain and sign in again
My token usage seems artificially high Check if an organization recently ran a bulk import and triggered automated report generations for hundreds of students simultaneously
