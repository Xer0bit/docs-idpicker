This guide covers advisor certificate operations and application document review.

## A) Certificates Workflow

1. In sidebar, open Certificates.
2. Review available certificate templates/items.
3. Open template/detail.
4. Configure issuance data as needed.
5. Send/issue certificate to selected student(s).
6. Confirm status (sent, pending, failed or equivalent).

## B) Application Documents Overview

1. In sidebar, open Application Documents.
2. Review student collections list.
3. Filter by status and review state.
4. Open a student collection detail.

## C) Review Student Documents (Click-by-Click)

1. Open specific step/category in student collection.
2. Open uploaded file metadata and preview/download.
3. Set review status:
   - Verified
   - Invalid
   - Pending (if re-evaluation needed)
4. If invalid, provide reason/comment.
5. Save review outcome.

## D) Collection-Level Status Updates

1. Review completion progress.
2. If complete and valid, move toward approval state.
3. If missing/invalid files remain, return with revision request.

## Common Advisor Questions

Q: Student uploaded file but not visible.
A: Refresh collection detail and verify step filter.

Q: Student asks why file was rejected.
A: Ensure invalid reason/comment is clearly written before save.

Q: Certificate action failed.
A: Retry issuance and validate template/student selection.
