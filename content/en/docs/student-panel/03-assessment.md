## Start an Assessment

1. Open Assessments from dashboard.
2. Begin questionnaire.
3. Answer all required questions.
4. Submit and wait for processing.

## Resume Interrupted Session

1. Return to Assessments.
2. Select Resume.
3. Continue from the last unanswered item.

## View Results

1. Open completed assessment card.
2. Review summary and details.
