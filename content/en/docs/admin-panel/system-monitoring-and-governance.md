## Status Module

1. Open Status page.
2. Review service health blocks.
3. Identify degraded components.
4. Correlate with recent alerts.

## Realtime Logs

1. Open Realtime Logs.
2. Start live stream view.
3. Filter by level/source if available.
4. Track suspicious error bursts.

## Free-Use Management

1. Open Free Use Management.
2. Review current quota/limit values.
3. Update limits according to policy.
4. Save and verify effect on applicable feature gates.

## Certificates and Application Documents

1. Open Certificates.
2. Review issuance queue/status.
3. Open Application Documents.
4. Review document validations and statuses.
5. Apply review decisions and notes.

## Common Questions

Q: Logs show spikes but dashboard looks normal.
A: Compare time windows and cross-check token usage metrics.

Q: Free-use update saved but behavior unchanged.
A: Verify environment and cache propagation delay.
